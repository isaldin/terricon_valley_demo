# Перфоманс без фреймворка: оптимизации, которые работают везде

Демо-проект для доклада на IT-конференции. Интернет-магазин электроники на Vue 3, проходящий через несколько итераций оптимизации — от baseline до полностью оптимизированного состояния.

## Стек

- Vue 3 (Composition API, `<script setup>`)
- Vue Router
- Vite
- Express (серверные оптимизации)

## Быстрый старт

```bash
npm install          # Установка зависимостей
npm run dev          # Dev-сервер
npm run build        # Продакшн-сборка
node server.js       # Express-сервер с оптимизациями
```

## Ветки-итерации

Каждая ветка — атомарная итерация оптимизации. Diff между ветками чистый и самодостаточный.

| Ветка | Тема |
|-------|------|
| `main` | Baseline без оптимизаций |
| `iter-1-resources` | Анализ бандла, code splitting, lazy loading |
| `iter-2-images` | Оптимизация изображений и шрифтов |
| `iter-3-rendering` | Рендеринг и визуальная стабильность |
| `iter-4-server` | Серверные оптимизации: HTTP/2, кеширование, сжатие |
